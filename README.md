<h2>🧐 Features</h2>

Here're some of the project's best features:

*   Add Delete Find and Delete User
*   Add Delete Find and Delete Book
*   Librarian and User roles
*   Find Book by Author
*   Find Book by Name
*   Show User's Book

*   Spring Web
*   Spring Data JPA
*   MySQL
*   Lombok
